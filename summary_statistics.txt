
# UAV-WSN Simulation Comprehensive Summary

**Note:** All scenario parameters and key metrics in this file reflect the current simulation state and are consistent with the latest scenario results and timings (see RESULTS_AND_DISCUSSION.md).

This summary lists all scenario parameter values and key simulation metrics (mean PDR, FND, LND, etc.) for each scenario (S1-A, S1-B, ..., S5-A, S5-B).

## Scenario Parameters (from omnetpp.ini)

- **N**: Number of sensor nodes
- **P**: Cluster head probability
- **v**: UAV speed (m/s)
- **E₀**: Initial node energy (J)
- **Packet Size**: Data packet size (bits)
- **Other**: As defined per scenario

## Key Metrics
- **Mean PDR**: Average Packet Delivery Ratio over all rounds
- **FND**: Round when First Node Dies
- **LND**: Round when Last Node Dies
